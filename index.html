box 렌더링

vw: viewport width 뷰포트 너비
ex) 100vw : 뷰포트(브라우져) 너비의 100% 를 사용
50vw : 뷰포트(브라우져) 너비의 50%를 사용

vh: viewport height 뷰포트 높이
ex) 100vh : 뷰포트(브라우져) 높이의 100% 를 사용
50vh : 뷰포트(브라우져) 너비의 50%를 사용

%: 부모요소의 상대적으로 계산되어진다.
